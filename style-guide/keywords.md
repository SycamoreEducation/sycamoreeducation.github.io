# Keywords

- Should always appear in lowercase

Incorrect: 
```php 
    $bar = TRUE;
```

Correct: 
```php 
    $bar = true;
```

---

Incorrect: 
```php 
    Global $variable;
```

Correct: 
```php 
    global $variable;
```

# Keywords

- Should always appear in lowercase
 - Incorrect: 
    ```php 
        $bar = TRUE;
    ```
 - Correct: 
    ```php 
        $bar = true;
    ```
 - Incorrect: 
    ```php 
        Global $g_user;
    ```
 - Correct: 
    ```php 
        global $g_user;
    ```
